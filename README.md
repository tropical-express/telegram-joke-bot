# Telegram Joke Bot
just a fun bot for when you feel sad and want to hear a joke.
---
# Features
- makes random jokes
- give you laughs
---
# Commands
- /start
- /joke
- /help
- /count
---
  # What does each command do
  - /start - start the bot
  - /joke - give you a random joke
  - /help - show this message
  - /count - count how many jokes you received
